Triangular has many options to configure the look / feel of the top toolbar used in the admin panel pages.

These options can be configured differently for each route, that way different pages in your admin panel can have different toobars.

For example this current view is using the default toolbar, this requires no additional code adding to your route because it is the default

{{'UI.TOOLBARS.SIZE' | translate}}
{{size.name}}

{{'UI.TOOLBARS.SIZE_SHOW' | translate}}

{{'UI.TOOLBARS.BACKGROUND' | translate}}

{{'UI.TOOLBARS.BACKGROUND_SHOW' | translate}}

{{'UI.TOOLBARS.SHRINK' | translate}}

{{'UI.TOOLBARS.SHRINK_SHOW' | translate}}

Example Route Code

.state('admin-panel.default.toolbars-default', {
        url: '/ui/toolbars/default',
        templateUrl: 'app/ui/toolbar-default.tmpl.html',
        data {
            extraClass: '{{codeVariables.extraClass}}',
            background: '{{codeVariables.background}}',
            shrink: {{codeVariables.shrink}}
        }
    })
